Getting Started

Langkah pertama menggunakan RepoFlow.

  • Buat Personal Access Token
  • Login dengan username dan token
  • Mulai kelola repository

Create Repository

Membuat repository baru di GitHub.

  • Klik menu "Create Repo"
  • Isi nama repository
  • Pilih Public/Private

Delete Repository

Menghapus repository dengan konfirmasi.

  • Klik menu "Delete Repo"
  • Masukkan nama repository
  • Konfirmasi nama
⚠️ Tindakan ini permanen!

Upload Mode 1: Multiple Files

Upload banyak file sekaligus.

  • Pilih tab "Mode 1"
  • Pilih target repository
  • Pilih file

Upload Mode 2: ZIP Archive

Upload ZIP, auto extract.

  • Pilih tab "Mode 2"
  • Upload file ZIP
  • Klik "Extract & Upload"

Live Terminal

Terminal real-time untuk melihat proses.

  • Muncul otomatis saat operasi
  • Warna log: Sukses, Error, Warning

Frequently Asked Questions

Bagaimana cara mendapatkan Personal Access Token (PAT)?
Buka GitHub Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Centang repo, delete_repo.
Apakah token saya aman?
Ya, token hanya disimpan di memori browser dan tidak pernah dikirim ke server manapun selain API GitHub.
